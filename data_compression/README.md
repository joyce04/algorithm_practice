실행 방법
- [encoding 프로그램 실행] python3 encoding.py <입력 파일> <출력 파일>
   예시 python3 encoding.py infile.txt encoding.txt
- [decoding 프로그램 실행] python3 decoding.py <입력 파일> <출력 파일>
   예시 python3 decoding.py encoding.txt outfile.txt
